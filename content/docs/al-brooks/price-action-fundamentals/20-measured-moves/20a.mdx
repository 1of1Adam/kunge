## 基本原理：第1段 = 第2段 (Leg 1 = Leg 2)

### 核心定义
测量目标位（Measured Move, 简称 MM）是市场中非常常见的现象。其基本逻辑是：
市场移动了一段距离（第1段），经历暂停或回调后，**第2段移动的距离往往与第1段大致相同**。

> **底层逻辑**：这是计算机算法（Computer Programs）主导的市场行为。当看到强势突破时，程序会寻找第2段趋势，并以第1段的高度作为首要目标。

### 嵌套结构 (Nested Structures)
市场结构具有分形特征，小结构可以嵌套在更大的结构中：
- 一个大的“第1段=第2段”走势中，其第2段内部可能包含更小级别的“第1段、回调、第2段”结构
- 这种嵌套关系帮助交易者在不同时间周期和波段级别上寻找共振的阻力位或支撑位

### 趋势强度的影响

| 市场状态 | 特征 | 预期 |
|:---|:---|:---|
| **强趋势** | 突破极其强势，几乎无回调 | 市场往往会忽略第一个MM目标，继续大幅延伸 |
| **深度回调** | 回调幅度深，类似震荡区间 | “第1段=第2段”通常只是**最低目标** |
| **通道化** | 跟随糟糕，动能减弱 | MM目标位往往标志着“急速变通道”的开始，随后演变为震荡区间 |

## 不同类型的测量方式

除了基本的波段复制（Leg 1 = Leg 2），还有其他几种常见的测量目标位计算方法：

### 基于突破高度 (Breakout Height)
当市场出现强势突破（如连续大阳线）时：
- **测量方法**：从第1根突破K线的开盘价，量度到最后1根突破K线的收盘价。
- **目标投影**：将这段高度向上（或向下）翻一倍。

### 基于震荡区间高度 (Trading Range Height)
当市场突破一个震荡区间时：
- **测量方法**：量度震荡区间的高度。
- **目标投影**：从突破点开始，投射等距离的高度作为目标位。

### 测量型缺口 (Measuring Gap)
在趋势中途出现的强劲突破（中继缺口）：
- **测量方法**：从趋势起涨点到缺口/突破点的高度。
- **目标投影**：将该高度向上翻一倍，寻找汇聚阻力位。

## 市场对目标位的反应机制

当价格接近或达到MM目标位时，市场参与者的行为会发生变化：

### 获利了结与逆势剥头皮
- **多头**：在上涨目标位处止盈（Take Profit）。
- **空头**：激进的空头开始做空剥头皮（Scalp），并准备在更高位置加仓。
- **结果**：买盘减少，卖盘增加，导致市场进入回调或震荡。

### 目标位失效（忽略目标）
> **信号**：如果市场毫不犹豫地大幅穿过某个MM目标位，说明计算机程序并不重视该目标。

**应对策略**：
1. 放弃当前目标位。
2. 寻找更远的目标（例如使用更大的波段作为第1段，或寻找更远的阻力位）。
3. 预期趋势将继续，直到遇到下一个阻力汇聚区。

### 80% 规则
在强上涨趋势中，**80%的反转尝试都会失败**。
- 即使在目标位出现反转形态（如双顶、高点抬高的大反转），大概率也会变成牛旗，随后趋势恢复。

## 处理起点的模糊性与不确定性

在实战中，确定“第1段”的起点往往存在主观性和模糊性。

### 常见的起点选择
1. **绝对极值**：K线图上的最低点/最高点。
2. **突破点**：震荡区间后的突破K线起点。
3. **更高的低点**：深度回调后的次级低点。

### 应对模糊性的策略

| 策略步骤 | 说明 |
|:---|:---|
| **1. 多重假设** | 同时标注几种可能的起点（如最低点、突破点、回调点），计算出不同的目标位 |
| **2. 优先近端** | 先参考最近、最明显的目标位。如果市场在该处有反应，说明算法认可该测量 |
| **3. 动态调整** | 如果价格突破了近端目标，则转向更远的目标（使用更早的起点或更大的波段进行测量） |
| **4. 观察共识** | 市场最终的反转点会告诉你，大部分计算机程序选择了哪个点作为起点 |

> **关键心态**：不需要精准预测哪一个是对的，而是观察市场在这些潜在区域的价格行为。

## 重要概念与关键点

| 概念 | 说明 |
|:---|:---|
| **算法主导** | 市场由计算机程序控制，它们倾向于使用数学上的对称性（如1:1的移动）来设定止盈点 |
| **汇聚 (Confluence)** | 当不同的测量方法（如波段测量、突破测量、区间测量）指向同一个价格区域时，该阻力/支撑位极其有效 |
| **事后清晰** | 很多时候，只有当市场在某个位置发生大反转后，交易者才能确定哪一段是真正的“第1段” |
| **止盈区域** | MM目标位主要用于**止盈**，激进者可用于**逆势剥头皮**，但不要指望它能直接引发大趋势反转 |

## 总结

本节课详细讲解了**测量目标位 (Measured Moves)** 的原理与应用：

1.  **核心模型**：最基础的模型是 **第1段 = 第2段**，即市场倾向于复制之前的波段幅度。
2.  **多样化测量**：除了波段复制，还可以利用**突破高度**和**震荡区间高度**进行翻倍测量。
3.  **交易逻辑**：MM目标位是多空双方博弈的节点——顺势者止盈，逆势者尝试剥头皮。
4.  **灵活性**：起点的选择具有模糊性，交易者应同时关注多个潜在目标，并根据市场的实际反应（是反转还是突破）来动态调整预期。如果一个目标被强势突破，立即寻找下一个更远的目标。
