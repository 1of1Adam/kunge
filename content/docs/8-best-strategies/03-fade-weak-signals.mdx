> **概率：** 60-80% 会反转到区间另一端或中点；目标是做反转剥头皮。
>
> **核心概念：** 在交易区间中，约 80% 的突破尝试都会失败。机构习惯用限价单低买高卖。交易者可以通过押注这种大概率的失败来获利——也就是去押注失败（Fade）那些注定失败的"强"信号。

## 理解交易区间

所谓交易区间，特征就是 **横盘震荡**，多空双方谁也没法完全控盘。这就好比一场拔河，谁也拉不动谁；价格就在确定的高低点之间来回拉锯，充满 **不确定性**。

价格行为中最基本的原则之一是：

> **在交易区间里，绝大多数突破尝试都会失败，概率高达 80%。**

市场经常猛烈冲击极点（比如区间的顶部或底部），诱骗交易者以为新趋势要来了，结果转头就跌回区间里。

---

## 什么是"弱信号"

所谓 **弱信号**，就是那些看似强劲、推向区间边缘，但注定要失败的行情：

| 表现形式 | 说明 |
|----------|------|
| **强趋势K线** | 看起来像是要突破，直冲区间极点 |
| **区间顶部的高1或高2** | 买入策略信号 |
| **区间底部的低1或低2** | 做空策略信号 |

新手常被这些信号 *套住*。看到顶部的大阳线就追涨，看到底部的大阴线就杀跌，结果眼睁睁看着市场反转。

然而，老手会把这些看作高潮或对极点的"测试"——既然大概率会失败，那就 **押注失败 (Fade)**。这就是机构在震荡市里的玩法：**低买高卖**。

## 入场策略与入场标准

### 在交易区间顶部

1. 盯着那些推向顶部的 **高1 或 高2 买入信号K线**
2. 如果这些信号带着 **十字星**、**长影线**，或者跟前面的 K 线 **重叠**，那就是 **弱** 信号
3. **入场**：在那根弱买入信号K线的高点或上方挂 **限价单做空**

![在弱信号K线上方设置限价单做空](/images/strategies/3_2.jpeg)

### 第二信号（胜率更高）

如果是 **第二信号**，胜率会高很多：
- 市场冲出新高（第一信号）
- 回调
- 再次冲击差不多的高点（第二信号）
- 形成微型双顶或更低的高点

这时候押注失败，胜算更大。

![在区间底部的弱空头K线处买入](/images/strategies/3_3.jpeg)

### 铁丝网形态

窄幅震荡，三根或更多 K 线重叠在一起，通常伴随十字星和长影线。

**入场**：等一根趋势K线突破后，准备用限价单 **押注失败**。比如，一根阳线向上突破，你就在它高点上方 1 个 tick 挂空单。

![在阳线上面挂限价卖单](/images/strategies/3_4.jpeg)

> **关键**：使用 **限价单**。你是逆势入场，要在反转发生前拿到更好的价格。

## 初始止损

用限价单逆势操作时，**保护性止损** 的位置很有讲究：

| 原则 | 说明 |
|------|------|
| **信号K线极点之外** | 止损放在信号K线高点上方（做空）或低点下方（做多）一段合理距离 |
| **允许过冲** | 市场经常会刺破阻力或未达支撑就反转，止损得给"过冲"留出余地 |
| **风险承受能力** | 如果止损宽了，就 **减小仓位**，保证单笔风险金额不变 |

## 交易管理

- **管理回调**：震荡市里回调是家常便饭，只要核心逻辑没变，别被小回调吓跑
- **加仓/减仓**：老手如果被套了一点，可能会在更好的价格加仓（**新手千万别碰**）
- **不要久留**：别指望拿着单子赌突破，到了目标位就赶紧跑

---

## 止盈

在交易区间极点押注弱信号失败，主要目标是 **剥头皮**。

| 目标 | 说明 |
|------|------|
| **区间另一端** | 最常见的目标，顶部做空就看底部 |
| **区间中心** | 很多时候，极点反转后会迅速回到区间中部 |
| **获利限价单** | 用获利限价单止盈，帮你克制贪婪 |

> **K线计数**：一笔成功的逆势单，可能会引发一段持续 10 根 K 线左右的两段式行情。

---

## 注意事项 / 何时避免

### 新手慎用

这属于"接飞刀"，风险大，需要经验。新手还是先学顺势的突破单入场。

### 绝对避免

| 情况 | 原因 |
|------|------|
| **强趋势** | 惯性会让 80% 的逆势尝试成为炮灰 |
| **区间中部** | 胜率只有 50% 左右，盈亏比不划算 |
| **窄幅震荡（铁丝网）** | 对新手来说就是绞肉机 |
| **强突破** | 连续几根大趋势K线，几乎没重叠，可能是真突破 |
| **情绪状态不好** | 情绪化交易 = 亏钱 |

---

## 判断胜率的背景线索

| 线索 | 含义 |
|------|------|
| **双向交易迹象** | 十字星、长影线、K线重叠，说明是震荡市，押注失败是好主意 |
| **缺乏后续** | 强 K 线冲向极点后没了动静，说明冲锋难以为继 |
| **市场惯性** | 交易区间倾向于维持震荡，突破大概率会失败 |
