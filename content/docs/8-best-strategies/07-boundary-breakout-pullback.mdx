> **胜率：** 60-70% 的概率会有后续行情；当多空双方都被套住时，胜率更高。
>
> **核心概念：** **突破回调**（即失败的反转）发生在反转尝试失败、市场恢复原有突破方向之时。被套的逆势交易者被迫平仓，为趋势延续**推波助澜**。这种"第二信号"机制让此类交易高度可靠。

## 如何判断突破强劲

在之前的章节中，我们讲过押注失败突破的高胜率交易。**关键在于如何判断**：何时预期突破会失败，何时预期突破会成功。

### 三个强劲突破的迹象

#### 1. 突破K线和急速的特征

| 特征 | 说明 |
|------|------|
| **大趋势实体** | 突破K线通常是强趋势K线，实体大，影线短或没有 |
| **紧迫感与距离** | 急速行情走得很远，强势突破了多个阻力或支撑位 |
| **成交量** | 可能伴随高成交量（例如平均水平的 10-20 倍） |
| **形成过程中回撤极小** | 理想情况下小于当前K线高度的四分之一 |

#### 2. 后续表现和随后的K线

| 特征 | 说明 |
|------|------|
| **立即出现后续行情** | 接下来的两三根K线也应该是趋势实体，大小至少达到近期平均水平 |
| **微型缺口** | 强阳线之后的K线低点，位于或高于前一根K线的高点 |
| **延续性** | 急速行情持续了 5-10 根K线，期间没有超过一根K线的回调 |

#### 3. 第一次回调的行为

| 特征 | 说明 |
|------|------|
| **回调滞后** | 第一次回调通常在突破发生三根或更多根K线之后 |
| **浅且短暂** | 仅持续一两根K线，且不是强反转K线 |
| **保持距离** | 没有跌回突破点，也没有触及保本止损 |

---

## 突破回调的本质

**突破回调**的特征是：逆着初始突破方向的小型、暂时性移动，通常持续一到五根K线，发生在原始突破后的几根K线内。

> 看到突破回调时，你的基本预期应该是：**原始突破趋势将恢复**。

突破 → 反转尝试失败 → 演变为突破回调 → 趋势恢复

这一序列是价格行为中最常见的形态之一，也被称为"**杯柄形态**"（Cup and Handle）。

---

## 入场策略和入场标准

### "失败的失败"（第二信号）

这是一个极强的入场策略：

1. 交易区间或强边界被强势突破
2. 紧接着出现逆势K线或小的逆势移动（"失败突破"）
3. 如果逆势移动**未能获得后续延续，反而被反转回原始突破方向**

> **例子**：向上突破交易区间后，出现了一根空头反转K线。如果随后价格涨破了这根K线的高点，空头的反转尝试就宣告失败，这就确立了一个突破回调的买入信号。

![多头突破回调："失败的失败"](/images/strategies/7_1.jpeg)

### 突破测试

发生在市场突破后，价格回调至（或接近）原始突破点或强边界时：

- 目的是"测试"发起突破的买家或卖家的力量
- 如果市场从该水平反弹并恢复原有方向，则确认了突破的强度
- 在确认测试成功的K线**外侧** 1 个 tick 处重新入场

### 强度比较

始终对比**初始突破的强度与随后反转尝试的强度**。如果初始突破很强，而反转尝试很弱，那么反转尝试很可能会失败。

### 在回调K线入场

| 交易类型 | 入场方式 |
|----------|----------|
| 多头突破 | 回调K线收盘后，在**其高点上方 1 个 tick 处挂买入突破单** |
| 空头突破 | 回调K线收盘后，在**其低点下方 1 个 tick 处挂卖出突破单** |

---

## 初始止损

| 交易类型 | 止损位置 |
|----------|----------|
| 买入入场 | 信号K线低点下方 1 个 tick，或回调结构的最低点下方 |
| 卖出入场 | 信号K线高点上方 1 个 tick，或回调结构的最高点上方 |

> 如果做大波段而非快速剥头皮，可能需要更宽的保护性止损，放在更重要的波段点下方。如果使用宽止损，记得降低仓位。

---

## 交易管理

### 移动止损

在多头趋势中，随着市场创出更高的高点，将止损上移至最近的更高低点下方。

> 注意不要太急于将止损移至保本，特别是在窄通道中。

### 加仓

对于经验丰富的交易者，突破回调提供了在趋势方向上加仓的机会。

### 适应

市场是动态的。如果市场行为暗示将转变为延长的交易区间，或者"Always-in"方向发生转变，准备好调整你的管理策略。

---

## 止盈

| 目标类型 | 说明 |
|----------|------|
| **测量移动** | 基于初始急速的高度，或发生突破的交易区间的高度 |
| **突破缺口** | 常充当测量型缺口，暗示市场可能走出等距行情 |
| **持续K线数量** | 一旦趋势恢复，预期行情将持续许多根K线，可能是几十根 |

---

## 注意事项 / 何时避免

| 情况 | 原因 |
|------|------|
| **强逆势反转** | 如果反转尝试比初始突破强得多，可能是真正的趋势反转 |
| **弱信号K线** | 十字星、实体小且影线长，表明缺乏信念 |
| **窄交易区间** | 突破单入场通常是必输的策略 |
| **缺乏清晰趋势** | 失败的突破导致真正反转的概率会增加 |
| **直面强支撑/阻力** | 当市场动能正在减弱时，风险很大 |
| **困惑或情绪化** | "如有疑虑，场外观望" |
