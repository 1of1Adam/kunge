> **概率：** 达到测量移动目标的概率在 60% 以上；在强突破中进行剥头皮，胜率可达 70-80%。
>
> **核心概念：** 强突破和急速，代表着紧迫感和信念。它们通常始于交易区间、现有趋势的回调之后，或者是反转失败。一旦突破K线看起来很强，顺势进场，胜率很高。

## 入场策略

### 突破的三种来源

| 来源 | 说明 |
|------|------|
| **交易区间** | 一旦从交易区间出现强突破，说明其中一方已经控盘 |
| **趋势恢复** | 在现有强趋势的回调之后，强突破意味着原始趋势的恢复 |
| **反转失败** | 反转尝试失败，变成顺势的旗形，导致顺着原趋势方向的强劲走势 |

## 入场标准

| 入场方式 | 说明 | 适用场景 |
|----------|------|----------|
| **市价单** | 直接市价买入（或做空），哪怕仓位很小 | 感觉到强烈紧迫感时 |
| **微小回调** | 在微小回调时入场（如 Emini 回调 1-2 个 tick） | 强趋势中 |
| **突破单** | 在前一根K线高点上方 1 个 tick 挂买入突破单 | 初学者最可靠的方式 |
| **K线收盘** | 如果强趋势K线让市场进入 "Always-in" 状态，在收盘时买入 | 确认趋势方向 |

> 激进的交易者可能在突破K线还没走完就进场了，但更安全的做法是等收盘后入场。

## 初始止损

**保护性止损**要放在"一旦被打掉，交易逻辑就不成立"的位置。

### 止损位置选择

- **急速底部**：如果是强多头急速，止损放在急速的低点，或急速底部那根K线的低点下方 1 个 tick
- **信号K线**：如果是基于信号K线入场，止损设在其低点下方（做多）或高点上方（做空）1 个 tick

> **重要**：因为初始止损可能比较宽，你要减小仓位，保证总风险金额在你的承受范围内。这能让你保持"无所谓"的心态，减少情绪干扰。

## 交易管理

### 加仓

老手可以在行情顺势发展时加仓；如果对趋势强度有信心，甚至可以在回调时加仓。

### 持仓度过回调

强趋势里的回调通常又短又小。你要拿得住单子，因为回调之后往往是趋势恢复。如果移动止损跟得太紧，很容易被"震出"局。

### 移动止损

有了浮盈后，可以移动保护性止损来锁定利润：
- 多头趋势创新高后，把止损移到最近的一个更高低点下方
- 在窄通道里要小心，别过早移到保本

## 止盈

| 目标类型 | 说明 |
|----------|------|
| **测量移动** | 幅度等于初始急速的高度，或被突破的交易区间的高度。强突破后，走到至少一个测量移动目标的概率通常在 60% 以上 |
| **部分止盈** | 接近重要阻力位或测量移动目标时分批止盈 |
| **阻力位** | 强急速往往会冲到明显的支撑或阻力位，然后反转 |

---

## 注意事项 / 何时避免

### 绝对不要交易

| 情况 | 原因 |
|------|------|
| **直冲强支撑/阻力** | 如果价格直冲重要价位且没有明确突破迹象，直接进场是不明智的 |
| **突破疲软** | 影线长、K线重叠严重、实体很小，或后续乏力 |
| **看不懂/不确定** | 市场很可能处于交易区间，突破容易失败 |
| **突破后紧接强反转** | 说明初始突破可能要失败了 |

### 特别警惕

- **窄交易区间**：这是突破单入场的噩梦，机构通常在看似突破的反方向做交易
- **陡峭通道的首次突破**：不要去交易逆势的首次突破
- **高潮衰竭**：长期趋势末端出现的最强突破，往往意味着衰竭而非强势

> **聪明的交易者会押注这些高潮失败。**

### 黄金法则

**不要逆 "Always-in" 方向交易**，除非有强劲、清晰的反转信号。市场有惯性，大多数反转尝试都会失败。
